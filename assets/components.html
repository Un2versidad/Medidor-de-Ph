Componentes del Sistema

Sensores para operación diaria en piscinas, granjas, pozos y tanques. Cada componente está integrado para lectura continua, calibración visual y guardado intencional de datos.

Arduino UNO Q + LED Matrix + LCD I2C

Controlador principal del sistema

Módulo que coordina la captura en sitio, la vista previa continua de sensores y el feedback visual de estado del agua.

Lectura activa Por comando "Tomar Lectura"
Vista previa Streaming local continuo
Feedback visual LED Matrix + LCD 16x2 + LED de estado
Integración Bridge con dashboard y backend

TDS Meter

Keyestudio TDS Meter

Medidor de sólidos disueltos totales (TDS) que cuantifica la concentración de sustancias disueltas en el agua. Esencial para evaluar la pureza y calidad del agua potable.

Rango de Medición 0 PPM a 1000 PPM
Precisión ±10% F.S.
Voltaje de Operación 3.3V - 5.5V
Salida Analógica 0-2.3V

Turbidity Sensor

Sensor de Turbidez

Sensor óptico que mide la turbidez del agua detectando partículas suspendidas. La turbidez es un indicador clave de la claridad del agua y posible contaminación.

Rango de Medición 0 NTU a 1000 NTU
Voltaje de Operación 5V DC
Salida Analógica 0-4.5V
Tiempo de Respuesta < 500ms

pH Sensor

Teyleten Robot DB7

Sensor electroquímico de pH que mide la acidez o alcalinidad del agua. El pH es un parámetro crítico para determinar la potabilidad y seguridad del agua.

Rango de Medición 0 pH a 14 pH
Precisión ±0.1 pH
Voltaje de Operación 5V DC
Tiempo de Respuesta < 1 minuto
